# CheXpert Dataset

This project uses the **CheXpert** chest X-ray dataset released by Stanford AIMI.

## Dataset Access
CheXpert requires agreeing to Stanford's data usage terms.

Download from:
https://aimi.stanford.edu/datasets/chexpert-chest-x-rays

## Expected Directory Structure

After downloading and extracting, place the files as follows:

data/raw/chexpert/
├── images/
│   └── CheXpert-v1.0/
│       ├── train/
│       └── valid/
└── metadata/
    ├── train.csv
    ├── valid.csv
    └── test.csv

⚠️ The dataset is NOT committed to this repository.
